# Wanderlust Bookings - Basic Trip Booking Website

A clean, responsive, and lightweight single-page template for a travel agency or trip booking website. Built using pure **HTML5**, **CSS3 (Grid/Flexbox)**, and **Vanilla JavaScript**, this template serves as an excellent foundational project or blueprint for travel web applications.

## ✨ Features
- **Modern & Responsive Design:** Uses a mobile-first approach with CSS Grid and media queries to look great on desktops, tablets, and smartphones.
- **Floating Interactive Form:** A clean visual hierarchy with input styling, dropdown menus, and hover states.
- **Smart Form Validation:** Built-in client-side JavaScript validation that ensures users cannot choose a return date that occurs before their departure date.
- **No External Dependencies:** Zero frameworks (No Bootstrap, Tailwind, or React). Pure, native web technologies for lightning-fast loading speeds.
